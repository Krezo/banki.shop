# Banki.shop App

## Stack

- Vue 2
- TypeScript
- Vuex 3 (State Management)
- Vue Router
- Domain-Driven Design
- ESLint + Prettier

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run linter
npm run lint

# Format code
npm run format
